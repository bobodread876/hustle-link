How it works

— three steps, done ✓

Your customers already order in the chat. Hustle Link just closes the money loop.

1
"3 box food, $800 each, drop off at Papine"

They order like always

Plain text, patois, shorthand — the agent reads the message and pulls out items, price, delivery spot, and who to call.

2
J$2,400 → 24,608 sats ⚡

Invoice drops in the chat

Total converts to sats at the live rate. The Lightning invoice lands right in the thread — payable from any wallet, any country.

3
ka-CHING ⚡ paid ✓

You get paid. Instantly.

Sats settle straight to your Flash wallet with the order in the memo. No chargebacks, no waiting on cash, no bank needed.

This receipt isn't a mockup.

That invoice was generated by the prototype against a live Flash lightning address — one command, no API keys, no signup. Every Flash username already is a Lightning address, so the money path needs zero custody and zero credentials. The buyer pays the vendor. Full stop.
